Sistema completo para gestión de contratos, órdenes de trabajo y rifas.

== Description ==

Sistema de Contratos es un plugin completo para WordPress que permite gestionar:

* Contratos de eventos
* Órdenes de trabajo
* Sistema de rifas
* Gestión de clientes
* Impresión de documentos

== Installation ==

1. Sube el plugin 'sistema-contratos' al directorio `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Configura el plugin en el menú 'Contratos' del panel de administración
